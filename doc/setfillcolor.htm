SetFillColor

SetFillColor(int r [, int g, int b])

Versione

1.3

Descrizione

Definisce il colore utilizzato per tutte le operazioni di riempimento (rettangoli riempiti e sfondo delle celle). Può essere espresso in componenti RGB o scala di grigi. Il metodo può essere chiamato prima che la prima pagina venga creata per mantenere il valore di pagina in pagina.

Parametri

r
Se g e b sono dati, indica il componente rosso; altrimenti indica il livello di grigio. Valori tra 0 e 255.
g
Componente verde (tra 0 e 255).
b
Componente blu (tra 0 e 255).
